## ?? Quick Tips

### Customize Colors
Edit `_Layout.cshtml`:
```html
<html lang="en" data-theme="corporate">  <!-- Change theme -->
```

### Update User Info
Edit `_Layout.cshtml` (line ~75):
```html
<span>Your Name</span>
<span>your.email@example.com</span>
```

### Add Avatar
Place image at: `/wwwroot/img/user-avatar.png`
(Or let it auto-generate fallback)

### Use Gradient Buttons
```html
<button class="btn bg-gradient-to-r from-blue-500 via-blue-600 to-blue-700 
         hover:bg-gradient-to-br text-white shadow-lg shadow-blue-500/50 border-0">
```
